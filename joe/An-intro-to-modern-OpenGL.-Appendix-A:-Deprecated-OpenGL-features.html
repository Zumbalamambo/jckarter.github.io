OpenGL is a driver API

OpenGL's primary purpose is to provide an interface to graphics hardware. In this role, it's more like assembly language than a high-level language—the OpenGL API gives you access to primitives like triangles, textures, buffers, and shaders that map to graphics hardware features, and leaves . Early 3d hardware accelerated a limited set of high-level operations, and early OpenGL was correspondingly higher-level, providing a range of fixed-function operations instead of programmable shaders.
